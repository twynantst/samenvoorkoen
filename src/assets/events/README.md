# Event Images

Deze folder bevat afbeeldingen voor externe acties die via Google Forms worden geregistreerd.

## Gebruik

1. Upload event afbeeldingen naar deze folder
2. In het Google Form, vul de bestandsnaam in (bijv: `quiz-avond.jpg`)
3. De afbeelding wordt automatisch getoond bij de actie
4. Klik op de afbeelding om deze te vergroten in lightbox

## Bestandsformaten

- Ondersteunde formaten: JPG, PNG, GIF, WEBP
- Aanbevolen breedte: 800-1200px
- Optimaliseer voor web (< 500KB per afbeelding)

## Naming convention

Gebruik beschrijvende namen zonder spaties:
- ✅ `kaas-wijn-avond.jpg`
- ✅ `benefiet-quiz-2026.png`
- ❌ `IMG_1234.jpg`
- ❌ `foto met spaties.jpg`
